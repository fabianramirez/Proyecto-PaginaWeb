HTML5 - Desarrollado por Tim Berners-Lee (fisico ) y Robert Cailliau (Ingeniero en sistemas Lanzado oficialmente en 1993 con el objetivo de poder programar páginas web. Primera estandarización del lenguaje estuvo a cargo de “Internet Engineering Task Force” Las normas y estándares de HTML son definidos por el organismo llamado “World Wide Web Consortium” (W3C), el cual lo define como “un lenguaje reconocido universalmente y que permite publicar información de forma global”. HTML se emplea para el desarrollo de diferentes aplicaciones especializadas como buscadores, tiendas en línea y banca electrónica.

Ventajas de Inline: Útil para hacer testing al inicio de un proyecto. Tiene un tiempo de recarga más rápido. Permite hacer quick-fixes con mayor facilidad.

Ventajas de instyle: Pueden ser leídos por todos los navegadores. Es imposible poner estilos a pseudo-elementos con inline. No necesita ser aplicado a cada elemento por individual.

SVG ANIMATION - Siglas de “Scalable Vector Graphics” es un formato de imagen 2D que guarda la información en forma de vectores (coordenadas) por lo que es posible utilizarlo fácilmente en web. Escalables. Pequeños. Crea tu .svg (illustrator es buena opción) Inserta .svg (código) en html. Comienza a animar!!
